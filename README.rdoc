Real time chat - RoR
=============================================

Real time chat running on Ruby on Rails.

This chat uses:

- MySQL and ActiveRecord
- Javascript, Jquery and ajax calls
- Ruby 2.1
- Rails 4
- css3
- html5

Gotchas:
- This app doesn't use migrations, db seeds, sofisticated sessions. We created it just for fun.
